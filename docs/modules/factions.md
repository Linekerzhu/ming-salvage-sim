# 党派模块

党派模块表示利益网络，不把任何一派写成纯善纯恶。党派影响大臣说什么、圣旨怎么执行、地方会不会抵制。

## 核心数值

- `影响力`：能影响多少人物、舆论、地方或军队。
- `亲疏`：对皇帝当前路线的支持程度。
- `利益受损`：最近是否被政策伤到核心利益。
- `反弹风险`：可能弹劾、拖延、串联或阳奉阴违的程度。

## 主要党派

- 太监/内廷宦官：皇帝家奴与内廷官僚，常见入仕路径，忠诚和执行力通常强，政治名望较弱。
- 阉党：近内廷、厂卫、司礼监和皇权的政治网络，包含外朝官僚与内廷人物；可制衡清流，也有独立自保和争权逻辑。
- 东林清流：重清议、制度和士林名分，可制衡阉党，但也会结党、护乡绅、拖旨或以公论压皇权。
- 宗室。
- 边镇武人。
- 江南士绅。
- 商帮海商。

## 玩法重点

圣旨触动党派核心利益时，即使皇威高，也可能被拖延、曲解或反扑。
太监、阉党、清流是朝局里互相牵引的三股力量。玩家更重要的目标是驾驭与重整边界，而不是按道德标签全盘清除某一方。
